# Project Template

## Overview
The `project` template is used for documenting projects, software, or other creations.

## Frontmatter

The following fields in the frontmatter require manual input:

*   `title`: `<title>` - The title of the project.
*   `date`: `<creation_date>` - The creation date.
*   `card.columnid`: `<column>` - The column ID for the board view.
*   `card.content`: `<short_description>` - A short description for the card.
*   `card.position`: `<position>` - The position of the card.
*   `card.title`: `<title (platform)>` - The title, optionally including the platform (e.g., "My App (iOS)").

## Body Content

The following sections after `---` require manual input:

*   `[summary:string]`: A brief summary.
*   `[download_link:string]`: Link to download the project.
*   `[platform:string]`: The platform the project is built for.
*   `[link:string]`: Link to the project's website or repository.
*   `[short_description:string]`: A short description.
*   `[description:md]`: Detailed description of the project (Markdown allowed).
